# Auto Polymers Website

This project is the static website for **Auto Polymers**, a manufacturer of precision moulded natural and synthetic rubber products based in Thrissur, Kerala, India.

## About Auto Polymers

Established in 1993, Auto Polymers specializes in high-quality rubber components for various demanding applications. With over 30 years of manufacturing experience, they offer:

- **Products**: O-Rings, Oil Seals, Gaskets, Grommets, Bush Kits, Diaphragms, Fabric Reinforced Pads, and Custom Moulded Parts.
- **Industries Served**: Automotive, Medical & Pharmaceutical, Industrial Machinery, Commercial, Agriculture, Marine, and Electrical & Electronics.
- **Commitment**: Safe, economical, durable, time-tested, reliable, and environmentally-friendly rubber engineering.

## Project Structure

This repository contains the front-end code for the static website.

- `index.html`: The landing page highlighting the company's core metrics, product ticker, filterable product grid, industries served, and unique differentiators.
- `company.html`: Detailed company profile, history since 1993, and mission/vision.
- `products.html`: Comprehensive catalog of all rubber components manufactured.
- `services.html`: Elaborates on the manufacturing and consulting services offered.
- `contact.html`: Contact form and location details.
- `style.css`: The global stylesheet containing the custom design system, modern aesthetics, animations, and responsive layouts.
- Images & Assets: Various PNGs used across the site (e.g., `Logo.png`, `kerala.png`, `factory.png`, `hero_bg.png`).

## Features

- **Modern & Responsive Design**: Clean aesthetics using HTML5 and CSS3, fully responsive across desktop, tablet, and mobile.
- **Interactive UI Elements**: Scroll-reveal animations, animated statistic counters, and a dynamic typing effect in the hero section.
- **Filterable Product Grid**: Users can easily filter products by industry sector (Automotive, Industrial, Medical, Commercial) directly from the home page.
- **Floating CTA & Back-to-Top**: Persistent navigation helpers for improved user experience.
